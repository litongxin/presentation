SpriteJS

A cross-platform lightweight 2D render object model

Key Features

  • Manipulate the sprites element as you do with the DOM elements
  • Similar box model and css attributes
  • Perform fast drawing with smart cache
  • Simple and powerful transition & animation API
  • Compatible with D3, Matter-js, Proton
  • Work with node-canvas, support Wechat MiniApps
